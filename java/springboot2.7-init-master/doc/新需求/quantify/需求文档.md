# 量化管理功能需求文档

## 1. 功能概述

量化管理功能是党建系统中的重要组成部分，主要用于对党员和党组织的各项活动进行量化统计和分析，提供数据可视化和报表导出功能，帮助管理员和党组织了解党建工作开展情况。

## 2. 核心功能

### 2.1 量化数据配置
- **指标管理**：支持新增、编辑、删除量化指标
- **指标配置**：为每个指标设置统计规则、适用维度和组织层级
- **状态管理**：支持启用/停用量化指标
- **批量操作**：支持批量删除指标
- **模板导入**：支持通过模板导入指标配置

### 2.2 统计报表查看
- **多维度统计**：支持按组织和个人维度进行统计
- **时间范围选择**：支持近3个月、近6个月、近1年和自定义时间范围
- **组织层级筛选**：支持按党委、支部等组织层级进行筛选
- **指标筛选**：支持按活动参与率、签到率、材料完成率等指标进行筛选
- **数据可视化**：支持柱状图、折线图、饼图等多种图表展示
- **数据明细**：展示详细的组织或个人数据
- **报表导出**：支持导出Excel格式的报表

## 3. 权限控制

- **管理员**：拥有全部功能权限，包括配置量化指标和查看所有报表数据
- **教师/培养联系人**：仅能查看所在支部和对接党员的相关数据
- **普通党员**：仅能查看个人相关数据

## 4. 界面设计

- **整体布局**：采用系统统一的卡片式布局，顶部为视图切换栏，中部为筛选和操作区，底部为数据展示区
- **响应式设计**：适配不同屏幕尺寸，在小屏幕设备上自动调整布局
- **交互体验**：提供直观的图表切换、数据筛选和详情查看功能
- **视觉风格**：采用系统统一的设计风格，使用Arco Design组件库

## 5. 数据来源

- **活动数据**：来自活动管理模块的活动参与和签到记录
- **材料数据**：来自发展阶段管理模块的材料提交记录
- **组织数据**：来自组织管理模块的组织架构信息
- **用户数据**：来自用户管理模块的用户信息

## 6. 性能要求

- **数据加载**：统计报表页面加载时间不超过2秒
- **图表渲染**：图表渲染时间不超过1秒
- **数据导出**：Excel报表导出时间不超过5秒（1000条数据以内）

## 7. 扩展性

- **指标扩展**：支持后续新增其他量化指标
- **维度扩展**：支持后续增加其他统计维度
- **图表类型**：支持后续增加其他图表类型
- **数据源扩展**：支持后续接入其他数据源
