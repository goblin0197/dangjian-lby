# 摘要

高校党建工作在党员发展流程跟踪、活动数据统计、材料归档管理等方面长期依赖人工操作和纸质文档，信息分散且难以实时共享，已无法满足跨校区多支部场景下的高效管理需求。针对上述问题，设计并实现了一套面向高校的智慧党建管理系统。系统采用前后端分离的B/S架构模式，后端以Spring Boot框架为核心整合MyBatis-Plus持久层与MySQL数据库，前端基于Vue 3响应式框架配合Arco Design Vue组件库构建用户界面，Redis缓存中间件负责会话存储与性能优化。系统围绕九大业务模块展开建设，其中党组织管理模块以树形结构维护党委至党支部的三级组织层级体系并支持负责人绑定；活动管理模块覆盖从创建发布到报名签到再到结束归档的完整生命周期流转；材料审核模块实现模板规范化管理与提交初审终审的多级审批流程；量化统计模块通过定时任务自动聚合个人参与度和组织活跃度等多维指标并以ECharts图表可视化呈现。经黑盒测试验证，各核心功能模块均按预期结果通过验收，系统具备支撑高校党建工作信息化转型的能力。

**关键词**：智慧党建；Spring Boot；Vue 3；前后端分离；量化统计

# Abstract

Party building work in colleges and universities has long relied on manual operations and paper-based documents in tracking member development processes, activity data statistics, and material archiving management. Information is scattered and difficult to share in real time, making it unable to meet the needs of efficient management across multi-branch scenarios on multiple campuses. To address these issues, a smart party building management system for colleges and universities was designed and implemented. The system adopts a front-end and back-end separated B/S architecture. The backend uses the Spring Boot framework as the core, integrating MyBatis-Plus persistence layer and MySQL database. The front-end is built based on the Vue 3 responsive framework with Arco Design Vue component library for user interface construction, while Redis cache middleware handles session storage and performance optimization. The system is constructed around nine business modules. The party organization management module maintains a three-level hierarchical system from party committees to party branches in a tree structure and supports leader binding. The activity management module covers the complete lifecycle from creation and publication to enrollment, check-in, and conclusion archiving. The material review module implements standardized template management and a multi-level approval process involving submission, initial review, and final review. The quantification statistics module automatically aggregates multi-dimensional indicators such as individual participation rates and organizational activity levels through scheduled tasks and presents them visually using ECharts charts. Verified through black-box testing, all core functional modules have passed acceptance according to expected results, demonstrating the system's capability to support the digital transformation of party building work in colleges and universities.

**Keywords**: Smart Party Building; Spring Boot; Vue 3; Front-end and Back-end Separation; Quantitative Statistics
